# MrDTwice

## Idea

MrDTwice is a lightweight community-driven web app for discovering, sharing, rating, and reviewing interesting places to visit.

The project is an MVP for an academy final exam and must be completed in about five days. The priority is to deliver a simple, working, and presentable product instead of a feature-heavy platform.

## Core Features

* Browse places
* Search and filter by text, category, country, and city
* View place details with images and average rating
* Add a new place with basic information and images
* Add ratings from 1 to 5 stars
* Add written reviews

## MVP Scope

Out of scope for the first version:

* User authentication
* User profiles
* Favorites and wishlists
* Social features
* Advanced moderation systems
* Recommendation algorithms

## Success Criteria

The MVP is successful if a user can browse places, search/filter them, open a detail page, submit a new place, upload at least one image, and leave a rating or review.
