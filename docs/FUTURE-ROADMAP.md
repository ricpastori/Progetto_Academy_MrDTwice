# MrDTwice — Future Roadmap & Product Backlog

Questo documento traccia la lista delle funzionalità, dei miglioramenti tecnici e delle evoluzioni di business identificate per le fasi successive al rilascio del Minimum Viable Product (MVP). L'organizzazione dei task segue le priorità di sviluppo decise dai 4 membri del team dell'Academy.

---

## Funzionalità Core & Ottimizzazioni Tecniche

- [ ] **Implementazione dell'autenticazione utente**
  - *Descrizione:* Sviluppo dei flussi di Sign-up, Login e gestione dei profili personali protetti.
  - *Ambito:* Full-Stack (Backend + Database + Frontend UI).

- [ ] **Ottimizzazione dell'accessibilità web**
  - *Descrizione:* Adeguamento dell'interfaccia e del markup HTML per garantire la piena conformità alle linee guida internazionali **WCAG**.
  - *Ambito:* Frontend UI/UX.

- [ ] **Integrazione di test automatizzati completi**
  - *Descrizione:* Copertura del codice tramite Unit Test e Integration Test per assicurare la stabilità della build e prevenire regressioni.
  - *Ambito:* DevOps / QA.

---

## Internazionalizzazione & Espansione del Mercato

- [ ] **Sviluppo della localizzazione in lingua inglese**
  - *Descrizione:* Configurazione dell'architettura multilingua e traduzione dell'interfaccia utente (Sito in ENG).
  - *Ambito:* Frontend.

- [ ] **Espansione geografica del catalogo**
  - *Descrizione:* Estensione del database relazionale per includere e categorizzare le regioni di tutto il mondo.
  - *Ambito:* Database / Architettura dei dati.

---

## Esperienza Utente & Community Features

- [ ] **Sistema di interazione e fidelizzazione**
  - *Descrizione:* Implementazione delle logiche per la gestione dei Preferiti, delle Wishlist e di un sistema di notifiche push/in-app per l'utente.
  - *Ambito:* Full-Stack.

- [ ] **Pannello di moderazione avanzata**
  - *Descrizione:* Creazione di un'interfaccia amministrativa dedicata al controllo, all'approvazione o alla rimozione dei contenuti inseriti autonomamente dalla community.
  - *Ambito:* Backend / Pannello Admin.

- [ ] **Trasformazione in Progressive Web App (PWA)**
  - *Descrizione:* Configurazione dei Service Worker e delle strategie di caching per permettere agli utenti di salvare il sito sulla home dello smartphone e consultare i luoghi salvati anche in modalità offline.
  - *Ambito:* Frontend / Mobile Optimization.

---

## Automazione AI & Modelli di Business (Monetizzazione)

- [ ] **Validazione Automatica delle Immagini via AI**
  - *Descrizione:* Integrazione di API esterne di Computer Vision per analizzare i media caricati in tempo reale dagli utenti, identificando ed escludendo in automatico file sfocati, non pertinenti o inappropriati.
  - *Ambito:* AI Integration / Backend layer.

- [ ] **Sezione "Esperienze" in Affiliazione**
  - *Descrizione:* Introduzione di un modello di monetizzazione per la piattaforma. Consente ad attività locali (guide turistiche, hotel, ristoranti) di inserire sponsorizzazioni mirate o sistemi di prenotazione diretta.
  - *Ambito:* Business Logics / API esterne.
